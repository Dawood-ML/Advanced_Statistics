# Advanced Statistics Using R

A lab repository documenting my hands-on journey through advanced statistical methods using R, following the curriculum of my 4th semester Advanced Statistics course.

## 📚 Primary Textbook
**Zhang, Z., & Wang, L. (2017). Advanced Statistics Using R.**
This is the main reference for all labs. The chapter structure of this repo follows the book directly.

## 📖 Secondary Textbook
**Understanding Advanced Statistical Methods (2013) — Using Excel.**
Used for conceptual understanding and statistical intuition before R implementation.

## 🛠️ Tools & Environment
- **Language:** R 4.5.2
- **IDE:** RStudio
- **Version Control:** Git & GitHub
